# 📱 Recibos Agua y Luz — PWA

## ¿Qué incluye?
- `index.html` — La app principal
- `manifest.json` — Configuración de la PWA (nombre, colores, íconos)
- `sw.js` — Service Worker (funciona offline)
- `icon-192.png` y `icon-512.png` — Íconos de la app (agrégalos tú)

---

## 🚀 Cómo subir a GitHub Pages (GRATIS, con HTTPS)

1. Ve a https://github.com y crea una cuenta si no tienes
2. Crea un repositorio nuevo (ej. `recibos-app`), que sea **público**
3. Sube los 4 archivos (index.html, manifest.json, sw.js, los 2 íconos)
4. En el repo ve a **Settings → Pages**
5. En "Source" selecciona **main branch / root**
6. GitHub te dará una URL tipo: `https://tuusuario.github.io/recibos-app/`

¡Listo! Abre esa URL en tu celular con Chrome y verás el botón "Agregar a pantalla de inicio".

---

## 📲 Cómo instalarla en el celular

### Android (Chrome):
- Abre la URL en Chrome
- Aparece un banner abajo que dice "Agregar a pantalla de inicio"
- O toca los 3 puntos → "Instalar app"

### iPhone (Safari):
- Abre la URL en Safari
- Toca el botón compartir (cuadro con flecha)
- "Añadir a pantalla de inicio"

## 🎨 Íconos
Necesitas crear dos PNGs:
- `icon-192.png` (192×192 px)
- `icon-512.png` (512×512 px)

Puedes generarlos gratis en: https://favicon.io o https://realfavicongenerator.net
